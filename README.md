# Zbot Releases

Private multi-device WhatsApp bot fleet APK releases.

## Latest Release

Download the latest APK from the [Releases page](https://github.com/Ali-123409/zbot-releases/releases).

## Verification

After downloading, verify integrity:
```bash
sha256sum Zbot-v1.0.0-debug.apk
# Should be: 2ea3b403ccc077b32aa7977fc78e47c10ec6f63119b19a18ed23e6923aa22972
```

## Install

1. Enable "Install unknown apps" for your file manager
2. Tap the APK to install
3. Open Zbot → Start Bot → Pair WhatsApp
